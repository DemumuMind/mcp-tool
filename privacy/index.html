Privacy

Privacy is simple because the site asks for very little.

The public site does not require accounts, private dashboards, or behavioral profiling to work.

Data collected

The marketplace primarily reads public source metadata and optional submission information. It is not designed as a logged-in product.

No marketplace profiles

The first release does not create user profiles, saved collections, or review histories.

External links

Many listings point to external docs, repositories, and product sites. Their privacy practices are separate from this marketplace.
