Logo Studio

Semantic product mining, encoder-aware prompt compilation, and diffusion logo generation.

Feed it a product name and description. The mining engine extracts differentiating metaphors, the compiler builds token-budgeted prompts, and pluggable backends generate brand marks without pushing the workflow into a cloud-only design stack.

Install

Package entry point

npm install @demumumind/logo-studio

Good for

  • Generating brand marks from a product description alone.
  • Switching between raw, studio, and README-driven prompt modes.
  • Respecting CLIP and T5 token budgets instead of blindly over-prompting.
  • Running against ComfyUI, Together AI, or local diffusers.

Not for

  • General image generation outside logo-specific workflows.
  • Typography-heavy brand systems with no human review.
  • Final identity work that skips designer QA.

Quick start

Basic flow

import { mine, compile, generate } from "@demumumind/logo-studio";

const mined = await mine({
  name: "Acme Corp",
  description: "We build bridges between legacy systems and modern cloud.",
  industry: "enterprise software",
});

const prompts = compile(mined, { promptMode: "studio" });

const result = await generate(prompts[0], {
  backend: "comfyui",
  seed: 42,
  width: 1024,
  height: 1024,
});

Pipeline

How it works

  1. Mine visual anchors and differentiators from the product description.
  2. Compile encoder-aware prompts with token budgets and negative prompts.
  3. Generate through ComfyUI, Together AI, or a local diffusers bridge.
